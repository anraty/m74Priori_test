Esimerkki

Ladataan data ja vhn ihmetelln. Luodaan samalla logTIAM -sarake tiamiinin logaritmia varten.

data_r <- read_xlsx("DATA/Finnish_M74_data-2022_paivitetty_TPa_21_01_2023.xlsx",
                  sheet = "Data")

data <- data_r %>% 
  select(YSFM = YSFM, TIAM = `TIAM_nmol/g`) %>% 
  filter(!is.na(YSFM) & !is.na(TIAM)) %>% 
  mutate(YSFM = YSFM/100,
         logTIAM = log(TIAM))

Katsellaan vhn milt nytt verrattuna logaritmoimattomaan tiamiiniin.

data %>%
  ggplot() +
  geom_point(aes(x = TIAM, y = YSFM), col = "red")+
  ggtitle("TIAM")

data %>%
  ggplot() +
  geom_point(aes(x = logTIAM, y = YSFM), col = "blue")+
  ggtitle("logTIAM")

Logaritmoitu nytt levenneen ja voisi toimia paremmin logisteisessa regressiossa.

Tutkitaan logTIAM akselia, jolla muutos 0-1 vlill tulee tapahtumaan.

data$logTIAM %>% summary
##     Min.  1st Qu.   Median     Mean  3rd Qu.     Max. 
## -2.80540 -0.03503  0.69320  0.54916  1.26273  2.56802

Noin -2.9 - 2.6 voisi olla kiinnostava vli sovittaa s-kyr. Luodaan simuloitu akseli kyseiselle vlille.

sim_x <- seq(-2.9, 2.6, by = 0.01)

Jotta s-kyr voidaan sovittaa tarvitaan expit-funktio ja rakenne logistista regressiota varten. Kirjoitan tss funktiona, jotta helpompi toistaa kokeillen erilaisia arvoja

expit <- function(x) 1/(1+exp(-x))

logReg <- function(a, b, x){
  expit(a + b * x)
}

Nyt alkaa arvaiu, kokeillaan ensin ihan raakasti a = 0 ja b = 1

#   sovitetaan todella raaka versio logististesta regressiosta 
#   ja plotataan saatu kyr

g1 <- logReg(a = 0, b = 1, x = sim_x)

plot(x = sim_x, y = g1, type = 'l',
     xlab = "logTIAM", ylab = "YSFM")

Ei nyt hyvlle. Ilmist tiedetn sen verran, ett tiamiini ja logtiamiini arvon kasvaessa, kuolleisuus pienenee. logistisen regression yhteydess tm tarkoittaa, ett b tulee olla negatiivinen.

Kokeillaan uutta arvausta

g2 <- logReg(a = 0, b = -1, x = sim_x)

plot(x = sim_x, y = g2, type = 'l',
     xlab = "logTIAM", ylab = "YSFM")
